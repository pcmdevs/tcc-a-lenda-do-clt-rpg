A Lenda do CLT: Uma Jornada de Texto ao Pygame
Este projeto foi desenvolvido como Trabalho de Conclusão de Curso (TCC). Trata-se de um RPG (Role-Playing Game) que explora a jornada de um trabalhador em um mundo de fantasia satírico, demonstrando a evolução de uma aplicação console para uma interface gráfica e sua exposição web.

Visão Geral
O projeto foi construído em três camadas principais para demonstrar diferentes competências técnicas:

Versão Core (Terminal): Foco em lógica de programação, estruturas de decisão complexas e manipulação de variáveis de estado do personagem.

Versão Gráfica (Pygame): Implementação de interface visual, manipulação de eventos de teclado/mouse e renderização de assets.

Landing Page (Web): Página estática para apresentação do universo do jogo e engajamento do usuário.

Tecnologias e Evolução
Fase 1: O Motor de Texto (Python Puro)
Lógica: Estruturas condicionais aninhadas para ramificação da história.

Interatividade: Input/Output dinâmico via console.

Destaque: Sistema de progressão de atributos do personagem.

Fase 2: Interface Gráfica (Pygame)
Biblioteca: Pygame.

Recursos: Criação de janelas, loop de eventos (Event Loop) e gerenciamento de estados de jogo (Menus vs. Gameplay).

Visual: Renderização de cores, fontes customizadas e controle de frames por segundo (FPS).

Fase 3: Apresentação Web (HTML/CSS)
Linguagens: HTML5 e CSS3.

Objetivo: Criar um "Showcase" do projeto, apresentando a lore (história) e as mecânicas para o público externo.

Funcionalidades do Jogo
Criação de Personagem: Escolha de classes e distribuição de pontos.

Sistema de Combate/Decisão: Escolhas que afetam o desfecho da narrativa.

Interface Intuitiva: Na versão Pygame, comandos visuais facilitam a experiência do usuário.

Narrativa Criativa: Ambientação baseada no cotidiano corporativo transposto para um cenário medieval/fantástico.

Estrutura dos Arquivos
A_LENDA_DO_CLT.py: Versão original em modo texto.

A_LENDA_DO_CLT_PYGAMES.py: Versão evoluída com interface gráfica.

Site_A_LENDA_DO_CLT.html: Página web de apresentação do projeto.

Como Executar
Pré-requisitos: Ter o Python instalado.

Instalação da biblioteca gráfica:

Bash

pip install pygame
Rodar a versão gráfica:

Bash

python A_LENDA_DO_CLT_PYGAMES.py
